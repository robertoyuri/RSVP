Instalar Python
Instalar Pycharm

Criar venv
pyhton -m venv venv

Entrar venv
source venv/bin/activate
venv/Scripts/activate

Instalar Django
pip install django

Iniciar o Projeto
django-admin startproject *nome_do_projeto_de_voces*

Entrar na pasta do projeto
cd *nome_do_projeto_de_voces*

Executar o projeto
python manage.py runserver

Iniciar aplicação
python manage.py startapp *nome_do_modulo_de_voces*

Criar pasta templates na raiz do projeto
Configurar a pasta templates no arquivo settings.py

Fazer migração do modelo de dados
python manage.py makemigrations

Excutar a migração
python manage.py migrate

criar um super usuario
python manage.py createsuperuser
